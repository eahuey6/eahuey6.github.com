Sum Arrow Function Challenge

This is a very short and simple function that just adds the two numbers and gives the sum. I took this example from the function examples assignment from DES157B. Link to arrow function examples and challenges from slides

Named Function

            function returnSum (num1, num2){
                var sum = num1 + num2;
                return sum;
            }

Function Expression

            const theSum = function(num1, num2){
                var sum = num1 + num2;
                return sum;
            }

Arrow Function

            const aVaraiable = (num1, num2) => num1 + num2;
