Feasibility Report

Features I Plan on Including

  • A form to enter in the affirmation
  • A gallery page to view other affirmations

  • A way to send the finished affirmations to other people through email

Technologies Required

  • Back4App for storing the data from the forms and HTML for creating the form
  • JavaScript and Back4App for retrieving data for the gallery
  • MagicMouse.js (cursor library) for designing the cursor

Though I do think that out of all the technologies, Back4App will take me the longest to work with and learn. I also am fairly skilled with using HTML and JavaScript for forms, since I worked on a project primarily using forms in a past class. I have not used MagicMouse.js, but I have used a different cursor library before (Kursor.js), and it was pretty easy to use so I should be able to figure out MagicMouse.js fairly fast.
